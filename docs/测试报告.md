- **权限测试**：以不同角色登录验证权限控制。
- **UI 测试**：验证页面渲染、导航跳转、表单验证提示、错误页面。

## 5. 已知问题与限制

### 5.1 已知问题

以下为当前版本中已知的轻微问题，不影响核心功能：

| 编号 | 问题描述 | 影响程度 | 计划修复版本 |
|------|----------|----------|-------------|
| KNOWN-01 | 任务看板暂不支持拖拽排序（Drag & Drop），需手动点击状态按钮更新 | 轻微 | V1.1 |
| KNOWN-02 | 日报编辑页面未限制教师修改学生日报内容（仅限制学生修改他人） | 轻微 | V1.1 |
| KNOWN-03 | 爬虫定时任务（APScheduler）需手动启动，未集成到容器启动流程 | 轻微 | V1.1 |
| KNOWN-04 | 系统日志（system_logs）记录接口已实现但未在前端提供全面查询页面 | 轻微 | V1.2 |
| KNOWN-05 | 部分表单提交后未返回上次浏览位置，而是回到列表第一页 | 轻微 | V1.1 |

### 5.2 功能限制

- 不支持 OAuth/SSO 第三方登录，仅支持本地账号密码登录。
- 不支持消息通知（站内信/邮件），任务变更需用户手动刷新查看。
- 图片上传仅支持服务端存储，未对接阿里云 OSS 等对象存储。
- 暂不支持 HTTPS，仅通过 HTTP 提供服务（生产环境建议配置 SSL 证书）。
- 爬虫模块仅支持静态 HTML 解析（requests + BeautifulSoup），不支持 JavaScript 渲染页面。

### 5.3 改进建议

1. 增加任务看板拖拽功能，提升用户体验。
2. 增加系统消息通知模块（WebSocket 或轮询）。
3. 增加批量导入/导出功能（Excel 格式）。
4. 增加教师端移动端适配。
5. 增加定时自动备份机制。
