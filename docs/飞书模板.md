# 飞书项目管理模板

> 以下内容直接复制到飞书多维表格使用

---

## 表一：项目任务表

**列字段：**

| 列名 | 类型 | 说明 |
|------|------|------|
| 任务编号 | 自动编号 | 系统生成 |
| 任务名称 | 文本 | 简洁描述任务内容 |
| 所属模块 | 单选 | 登录/项目中心/小组管理/任务看板/日报/Bug管理/爬虫/数据可视化/AI记录/部署 |
| 负责人 | 人员 | @相关成员 |
| 开始日期 | 日期 | 任务开始日期 |
| 截止日期 | 日期 | 任务截止日期 |
| 优先级 | 单选 | 高 / 中 / 低 |
| 状态 | 单选 | 待开始 / 进行中 / 待测试 / 已完成 / 已延期 / 已关闭 |
| 预计工时(h) | 数字 | 预估工时 |
| 实际工时(h) | 数字 | 实际工时 |
| 完成说明 | 多行文本 | 完成情况的描述 |
| 关联代码提交 | 链接 | GitHub commit 链接 |
| 关联文档 | 链接 | 相关飞书文档链接 |
| 教师评价 | 多行文本 | 教师反馈 |

---

## 表二：每日进度表

**列字段：**

| 列名 | 类型 | 说明 |
|------|------|------|
| 日期 | 日期 | 日报日期 |
| 学生姓名 | 人员 | @学生 |
| 今日完成 | 多行文本 | 今天完成的具体内容 |
| 今日问题 | 多行文本 | 遇到的问题和困难 |
| 明日计划 | 多行文本 | 明天的计划工作 |
| AI工具使用情况 | 多行文本 | 使用了哪些AI工具，解决了什么问题 |
| 代码提交链接 | 链接 | GitHub commit URL |
| 截图 | 图片 | 完成内容的截图 |
| 自评完成度 | 数字(1-5) | 1=很差 5=很好 |
| 教师点评 | 多行文本 | 教师反馈 |

---

## 表三：Bug 跟踪表

**列字段：**

| 列名 | 类型 | 说明 |
|------|------|------|
| Bug编号 | 自动编号 | 系统生成 |
| Bug标题 | 文本 | 简洁描述问题 |
| 所属模块 | 单选 | 登录/项目中心/小组管理/任务看板/日报/Bug管理/爬虫/数据可视化/AI记录/部署 |
| 提出人 | 人员 | @提出者 |
| 负责人 | 人员 | @修复人 |
| 严重程度 | 单选 | 致命 / 严重 / 一般 / 轻微 / 建议 |
| 状态 | 单选 | 新建 / 已确认 / 修复中 / 已修复 / 已关闭 / 暂不处理 |
| 复现步骤 | 多行文本 | 详细的复现步骤 |
| 解决方案 | 多行文本 | 修复方法和说明 |
| 截图 | 图片 | Bug 截图 |
| 关闭时间 | 日期 | Bug 关闭日期 |

---

## 表四：AI 使用记录表

**列字段：**

| 列名 | 类型 | 说明 |
|------|------|------|
| 日期 | 日期 | 使用日期 |
| 使用人 | 人员 | @使用者 |
| 工具名称 | 文本 | 通义灵码/豆包/ChatGPT等 |
| 使用场景 | 单选 | 数据库设计/Flask路由/报错解释/样式优化/测试用例/Dockerfile/docker-compose/部署文档/日志分析/项目总结/其他 |
| 提示词 | 多行文本 | 输入的提示词内容 |
| AI输出摘要 | 多行文本 | AI返回的关键内容 |
| 是否采纳 | 复选框 | 是/否 |
| 人工修改内容 | 多行文本 | 对AI输出的修改 |
| 最终效果 | 多行文本 | 实际效果评估 |
| 风险说明 | 多行文本 | 可能的风险和注意事项 |

---

## 表五：项目风险表

**列字段：**

| 列名 | 类型 | 说明 |
|------|------|------|
| 风险编号 | 自动编号 | 系统生成 |
| 风险描述 | 多行文本 | 风险的具体描述 |
| 影响程度 | 单选 | 高 / 中 / 低 |
| 发生概率 | 单选 | 高 / 中 / 低 |
| 负责人 | 人员 | @责任人 |
| 应对措施 | 多行文本 | 应对策略 |
| 当前状态 | 单选 | 活跃 / 已解决 / 已忽略 |
| 更新时间 | 日期 | 最后更新日期 |

---

## 表六：项目验收表

**列字段：**

| 列名 | 类型 | 说明 |
|------|------|------|
| 验收项 | 文本 | 验收内容描述 |
| 验收标准 | 多行文本 | 量化的验收标准 |
| 是否完成 | 复选框 | 是/否 |
| 截图证明 | 图片 | 完成的截图 |
| 代码链接 | 链接 | 相关代码 |
| 文档链接 | 链接 | 相关文档 |
| 教师评分 | 数字 | 分数 |
| 教师备注 | 多行文本 | 补充说明 |
